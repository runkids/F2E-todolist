# todolist

The F2E - 前端修練精神時光屋 第一週 todolist

`npm install`

`npm start`

`npm run build`

open with  
`http://localhost:8090/f2e/week1/index.html`


## User Story
1.能夠紀錄每天的待辦事項
2.可標示每個待辦優先重要級別，預設為無。
3.可拖拉待辦事項，調整排序。
4.每筆待辦可新增時間提醒 (yyyy/mm/dd hh:mm)
5.每筆待辦可再填寫評論與附加檔案
6.待辦狀態：全部顯示(預設)、待處理、已處理
7.待辦事項過多時，需考量內容是否需要折疊
